# 🔔 텔레그램 알림 설정 가이드

RenderTracker 알림을 텔레그램으로 받기 위한 설정 방법입니다.

---

## 1. 내 고유 ID(Chat ID) 확인 🆔

1.  **봇 접속**: [내 ID 확인 봇 (@userinfobot)](https://t.me/userinfobot) 링크 클릭
2.  **대화 시작**: 하단의 **[시작(Start)]** 버튼 클릭
3.  **ID 복사**: 나타나는 정보 중 **`Id: 123456789`** 형태의 숫자를 복사하여 메모

---

## 2. 나만의 알림 봇 생성 (Token 발급) 🤖

1.  **봇 관리자 접속**: [봇 관리자 (@BotFather)](https://t.me/botfather) 링크 클릭
2.  **새 봇 생성**: 채팅창에 **`/newbot`** 입력
3.  **이름 설정**: 봇의 이름 입력 (예: `RenderTracker`)
4.  **아이디 설정**: 봇의 아이디 입력 (반드시 `_bot`으로 종료, 예: `rendertracker_bot`)
5.  **토큰 복사**: 성공 메시지에서 **`Use this token to access the HTTP API:`** 뒤의 **`토큰(Token)`**을 클릭하여 복사
6.  **봇 활성화**: 직접 만든 봇 링크(예: `t.me/봇아이디_bot`) 접속 후 **[시작(Start)\)** 클릭 (**필수!**)

---

## 3. RenderTracker 앱 적용 ⚙️

1.  **텔레그램 아이콘 활성화**: 앱 상단의 텔레그램 아이콘 활성화(컬러)
2.  **설정 열기**: 앱 우측 상단의 **톱니바퀴 아이콘** 클릭
3.  **정보 입력**: **[텔레그램 Bot]** 섹션의 **[Bot Token]**과 **[Chat ID]** 칸에 복사한 정보 입력

---

## ⚠️ 주의사항
*   발급받은 **Bot Token**은 본인 봇의 비밀번호와 같습니다. 타인에게 노출되지 않도록 주의하세요.
